## 配置校验规则

所有 37 个配置项在保存时按以下规则校验，校验失败时管理端实时提示且不写入数据库。

| 配置键 | 类型 | 合法范围 | 非法值处理 |
|--------|------|---------|-----------|
| borrow_limit | int | 1–50 | 拒绝保存并提示 |
| borrow_period_days | int | 1–90 | 拒绝保存并提示 |
| overdue_fine_per_day | decimal | 0–100 | 拒绝保存并提示 |
| lost_book_fine_multiplier | decimal | 1.0–3.0 | 拒绝保存并提示 |
| renewal_discount / multi_child_discount | decimal | 0.5–1.0 | 拒绝保存并提示 |
| member_days / observation_days | int | 1–730 | 拒绝保存并提示 |
| member_grace_days | int | 0–90 | 拒绝保存并提示 |
| *\_remind_days / due_remind_days | 逗号分隔 int 列表 | 每项 0–30，自动去重排序 | 格式错误拒绝保存 |
| price\_\* | decimal | 0.01–100000 | 拒绝保存并提示 |
| deposit_amount | decimal | 0–10000 | 拒绝保存并提示 |
| reservation_expire_hours | int | 1–168 | 拒绝保存并提示 |
| order_expire_minutes | int | 5–120 | 拒绝保存并提示 |
| vocab_lookup_limit / trial_pages | int | 0–1000 | 拒绝保存并提示 |
| quiz_pass_rate | decimal | 0.5–1.0 | 拒绝保存并提示 |
| quiz_total_questions / quiz_pass_count | int | 1–20 | 拒绝保存并提示 |
| quiz_cooldown_minutes | int | 5–1440 | 拒绝保存并提示 |
| bookshelf_limit | int | 0–500 | 拒绝保存并提示 |
| activity_cancel_hours | int | 1–72 | 拒绝保存并提示 |
| checkin_min_minutes / checkin_min_vocab | int | 1–120 | 拒绝保存并提示 |
| daily_checkin_limit | int | 1–10 | 拒绝保存并提示 |
| enable\_\* | bool | true/false | 非 bool 值拒绝保存 |

未列范围的配置键按上述同类规则执行。所有校验失败在管理端实时提示，不写入数据库。
